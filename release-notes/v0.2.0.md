# v0.2.0

## Highlights
- Added immutable tree operations: rename, move, and copy.
- Added path-safe descendant updates for folder operations.

## Changes
- Added `renameResourceInTree(resources, targetPath, newName)`.
- Added `moveResourceInTree(resources, targetPath, destinationFolderPath)`.
- Added `copyResourceInTree(resources, targetPath, destinationFolderPath, options?)`.
- Expanded test coverage from 9 to 13 tests.
- Updated README with new API usage and examples.

## Breaking Changes
- None.

npm install @shahrukh404/file-explorer@0.2.0
